# 🤖 AI Learning Chatbot

This is my first AI chatbot built with **n8n** using an **AI Agent**, **OpenAI Chat Model**, and **Simple Memory**.

I built this project while following the **Kamran AI Insight AI Automation Bootcamp** to practice building AI workflows and better understand how AI agents work.

## ✨ Features

- Answers user questions using OpenAI
- Remembers previous messages with Simple Memory
- Built using a no-code workflow in n8n

## 🛠️ Technologies Used

- n8n
- OpenAI
- AI Agent
- Simple Memory
